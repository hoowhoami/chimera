设计理念

Chimera 框架的设计灵感来自 Spring Boot，旨在为 Rust 开发者提供一个熟悉且强大的 Web 开发体验。

我们相信：

  • 约定优于配置 - 提供合理的默认配置，减少样板代码
  • 开发者友好 - 清晰的 API 设计和完善的文档
  • 类型安全 - 充分利用 Rust 的类型系统保证代码质量
  • 高性能 - 基于 Tokio 和 Axum 构建，提供出色的性能
